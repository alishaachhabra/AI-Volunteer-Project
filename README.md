# 🌍 AI Volunteer Impact Platform

An AI-based platform that connects volunteers with NGOs and provides smart chatbot guidance.

---

## 🚀 Features

- 🔍 Find NGOs based on location, work, and eligibility
- 🤖 AI chatbot to answer questions about NGOs
- 📊 Dataset-based NGO filtering
- 🧠 Smart search combining AI + data
- 📥 Download results as CSV

---

## 🛠 Technologies Used

- **Streamlit** → for building web app UI  
- **Pandas** → for data handling  
- **Requests** → for API communication  
- **Openpyxl** → for reading Excel dataset  

---

## 📂 Dataset

Contains NGO details like:
- Name  
- Location  
- Work Area  
- Eligibility  

---

## 💡 Use Case

Helps users:
- Find NGOs easily  
- Understand volunteering opportunities  
- Get instant AI guidance
